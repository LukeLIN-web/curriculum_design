## 渐进符号

大O符号上界 T(N) = O(f(N)) T(N) <= c * f(N)
存在c 使得   c*fn > Tn   
大Ω符号 T(N) = Ω(f(N)) T(N) >= c * f(N)是下界

大Θ符号 T(N) = Θ(f(N)) T(N) = O(f(N)) && T(N) = Ω(f(N)) 是上界又是下界

小o符号 T(N) = o(f(N))   T(N)< c * f(N)

* T1(N) + T2(N) = max(O(f(N)), O(g(N))

* T1(N) * T2(N) = O(f(N) * g(N))

* 若 T(N) 为 k 次多项式，则 T(N) = Θ(N ^ k)

* log(k, N) = O(N)
